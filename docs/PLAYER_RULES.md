# VRMine ポップルールブック

## 目的
謎の地下鉱脈で波を撃ち込み、仲間より早く盤面の配置を完全一致で言い当てよう！

## セットアップ
- 1テーブル＝10×8マスの盤面、周囲36箇所が波の入口。
- プレイヤーは同じログボードを共有。カラフルなビー玉が波の色、矢印が出口IDだ。
- 90秒タイマーをセットして、スタートプレイヤーを決めよう。

## あなたのターン
1. 好きな入口に波を発射（コントローラで指差しorボタン）。
2. 波はまっすぐ進み、色セルで90°ターン、黒セルで吸収される。
3. 出口ID・波の色・フラグがログボードに追加されるので、全員でチェック。
4. まだ確信がなければ次の人へ。盤面を全員で推理しよう。

## 完全一致宣言
- 配置がわかったら「完全一致！」と宣言して回答。
- 正解なら勝利。外したら次のターンまで口出し禁止＆1ターン休み。

## ログの読み方コツ
- 同じ入口で色が変わったら、その先に色セルが潜んでいるサイン。
- 黒セルに吸われた波はログで「×」マーク。配置候補を即座に消そう。
- 連続で入口を変えて三角形を描くように試すと、曲がり角の位置が浮かび上がる。

## ショートヒント
- 迷ったらチームで「出口読み合わせタイム」を作ると推理スピードが爆上がり。
- メモは入口IDと出口IDのペアを書くだけでも十分効果あり。
- ミスしてもOK！派手に波を撃って盤面をカラフルにしよう。
