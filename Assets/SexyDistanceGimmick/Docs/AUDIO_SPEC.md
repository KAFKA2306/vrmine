# 音声ファイル仕様

## 概要

SexyDistanceGimmick に必要な音声ファイル一覧と生成指示。

---

## 必要ファイル（計7点）

| ファイル名 | 用途 | 長さ | ループ |
|------------|------|------|--------|
| `Whisper_Default.wav` | 接近時の基本囁き | 10-30秒 | ✅ |
| `Whisper_Chest.wav` | 胸接触時 | 5-10秒 | ✅ |
| `Whisper_Neck.wav` | 首接触時 | 5-10秒 | ✅ |
| `Whisper_Ear.wav` | 耳接触時 | 5-10秒 | ✅ |
| `Whisper_Waist.wav` | 腰接触時 | 5-10秒 | ✅ |
| `Whisper_Thigh.wav` | 太腿接触時 | 5-10秒 | ✅ |
| `Touch.wav` | 接触SE | 0.3-1秒 | ❌ |

---

## フォーマット

```
形式: WAV または OGG
サンプルレート: 44100Hz
ビット深度: 16bit
チャンネル: モノラル推奨（3Dサウンド用）
```

---

## AI音声生成用プロンプト

### 囁き共通指示

```
【キャラクター】
- 性別: 女性
- 年齢: 20代前半
- 声質: 低めのウィスパーボイス、息遣い多め
- 感情: 挑発的、余裕がある、少し楽しんでいる

【技術指示】
- マイクに極めて近い距離で収録したような質感
- リップノイズ、吐息を自然に含める
- BGM・効果音なし、音声のみ
- ループ可能な終わり方（フェードアウトまたは自然な区切り）
```

### Whisper_Default（基本囁き）

```
【状況】
相手との距離を詰めながら、耳元で囁いている。
まだ触れてはいない。距離が近づくにつれて声のトーンが下がる。

【セリフ例】
「ねえ…逃げないの？」
「もう少し…近くに行っても、いい？」
「ふふ…心臓の音、聞こえそう」
（吐息のみ）
「このまま…ずっと…」
```

### Whisper_Chest（胸接触）

```
【状況】
胸に触れられた瞬間の反応。驚きと快感が混在。

【セリフ例】
「っ…そこ…」
「大胆…ね」
「ん…もっと…触って」
```

### Whisper_Neck（首接触）

```
【状況】
首筋に触れられた。敏感な場所への接触で声が震える。

【セリフ例】
「あ…首…弱いの…」
「やだ…くすぐったい…」
「んっ…」
```

### Whisper_Ear（耳接触）

```
【状況】
耳に直接触れられた。最も敏感な反応。

【セリフ例】
「耳…だめ…っ」
「そこ…一番…っ」
（荒い吐息のみ）
```

### Whisper_Waist（腰接触）

```
【状況】
腰に手を回された。支配的な接触への反応。

【セリフ例】
「捕まっちゃった…」
「もう…逃がしてくれないの？」
「ん…そのまま…」
```

### Whisper_Thigh（太腿接触）

```
【状況】
太腿に触れられた。際どい場所への接触で緊張。

【セリフ例】
「そこは…っ」
「いじわる…」
「もっと…上？」
```

### Touch.wav（接触SE）

```
【音の種類】
- 柔らかい衝撃音（肌と肌が触れるような音）
- または短い息を飲む音「はっ」
- 0.3〜1秒

【参考】
- 布が擦れる音 + 吐息
- 低いバスドラムのようなソフトインパクト
```

### 追加：Yobai Mode 専用ボイス

```
【状況設定】
深夜。寝ている相手の部屋に忍び込み、枕元に座って顔を覗き込んでいる。
相手が起きていることには気づいているが、あえて無視して強行する。

【Whisper_Yobai_Approach.wav】
「…起きてるんでしょ？」
「ふふ…寝たふりしても、だめ」
「鍵、開いてたよ…」

【Whisper_Yobai_Ear.wav】
「今夜は…帰らないから」
「…声、出しちゃだめ」
（衣擦れの音）
```
