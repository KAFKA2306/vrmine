# 実装完了チェックリスト (Definition of Done)

実装作業完了後、以下の項目を実機（VRChat Client）およびエディタで確認してください。

---

## 1. 動作検証 (Functionality)

### 距離・移動
- [ ] **自動起動**: TriggerDistance (1.5m) に入ると自動的にゴーストが動き出すか
- [ ] **追従**: プレイヤーの頭部に向かって直線的に移動するか
- [ ] **停止**: MinDistance (0.35m) で正確に停止し、それ以上めり込まないか
- [ ] **永続**: プレイヤーが動いても追従し続け、勝手に戻ったり消えたりしないか

### 音響 (Audio)
- [ ] **距離減衰**: 距離が近づくにつれて、囁き声（Default）が大きくなるか
- [ ] **接触音**: 接触した瞬間、SE が一度だけ鳴るか
- [ ] **接触ボイス**: 接触部位に対応した正しいボイス（耳、首など）に切り替わるか
- [ ] **ボイス保持**: 接触後、少しの間（2秒）最大音量が維持されるか

### 接触判定 (Touch)
- [ ] **部位識別**: 胸、首、耳、腰、太腿のそれぞれで反応するか
- [ ] **クールダウン**: 連続して触れたとき、音が連打されずに適切な間隔（0.35秒）があるか
- [ ] **優先順位**: 新しい部位を触ったとき、即座に反応が切り替わるか

### アニメーション
- [ ] **前傾 (Lean)**: 近づくにつれて（または接触時に）ゴーストが前傾姿勢になるか
- [ ] **待機**: プレイヤーが遠くにいるときは直立（Idle）に戻るか
- [ ] **Yobai (Option)**: 這いずり（Crawl）アニメーションが正しく再生されるか（Yobai Mode 時）

### ハプティクス (VR Only)
- [ ] **振動**: 接触した方のコントローラーが振動するか
