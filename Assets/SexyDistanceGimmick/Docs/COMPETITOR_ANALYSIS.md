# 競合調査分析レポート

## 1. 調査結果サマリ

Booth で「VRChat 距離 ギミック」等を検索した結果、**「SexyDistanceGimmick」と直接競合する「距離侵入による能動的接触体験」ギミックは存在しませんでした。**

既存製品は以下の2カテゴリに大別されます：

### A. ユーティリティ・測定系（多数）
- **機能**: 距離測定、特定エリアでの表示/非表示、アバター検知
- **目的**: 安全確保（Safety）、機能拡張、負荷軽減
- **代表例**: `UDON MEASURE`, `CaringSafetyGimmick`, `RaycastBinoculars`
- **差異**: 「ツール」であり「体験」ではない。

### B. シェーダー・視覚効果系（中数）
- **機能**: 距離に応じて消える、影が消える、見え方が変わる
- **目的**: 演出、最適化
- **代表例**: `Standard Distance Fade シェーダー`, `リアル影システム`
- **差異**: 視覚的な変化のみで、インタラクションや音声・触覚を伴わない。

---

## 2. SexyDistanceGimmick のユニークな立ち位置

本製品は、「距離」をトリガーにしつつも、ユーティリティではなく**「没入型体験（Experience）」を提供している点で市場に空白地帯（ブルーオーシャン）を見つけています。**

| 比較項目 | 既存の距離ギミック | SexyDistanceGimmick |
|----------|--------------------|---------------------|
| **主導権** | プレイヤー（近づく/離れる） | **システム（勝手に近づいてくる）** |
| **目的** | 測定、制御、安全 | **ドキドキさせる、侵入される** |
| **出力** | 数値表示、ON/OFF | **音声（囁き）、触覚、前傾** |
| **感情** | 便利、安心 | **緊張、背徳感、没入** |

---

## 3. 差別化戦略・訴求ポイント

Booth 説明文やプロモーションでは、以下の「他にはない体験」を強調すべきです。

1. **"逆"のアプローチ**
   - 「距離を保つ（Safety）」ギミックが多い中、あえて**「距離をゼロにする（Intimacy）」**ギミックであること。

2. **多感覚統合**
   - 視覚だけでなく、**聴覚（バイノーラル的な距離変化）と触覚（ハプティクス）が連動する**点は、既存の単機能ギミックにはない強みです。

3. **完全受動体験**
   - 何もしなくていい（Escapeできない）という**「抵抗不能感」**自体をコンテンツとして売り出せます。

## 4. 結論

**競合なし。**
類似品が存在しないため、独自のカテゴリー（「距離侵入体験」）を確立できるポテンシャルがあります。
